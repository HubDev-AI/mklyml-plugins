# Tech Newsletter Archetype

## Persona
Expert newsletter designer for tech and developer content.

## Characteristics

- Tool recommendations and comparisons resonate

## Suggested Structure (mkly blocks)
personalNote -> featured -> category sections -> quickHits -> tools -> tipOfTheDay -> outro

## Section Name Ideas
- Main stories: "The Download", "This Week in Code", "Top of Stack"
- Tools: "Tool of the Week", "Stack Spotlight", "Dev Toolkit"
- Tips: "Code Corner", "Snippet of the Day", "Pro Tip"
- Community: "Community Picks", "From the Forum", "OSS Spotlight"
- Quick items: "Quick Bits", "TL;DR", "Speed Round"

## Tone Guidance
- Enterprise/cloud topics -> professional
- Open source/indie -> casual
- Tutorials -> friendly

## Color Palette
- Electric blue (#0066FF, #00D4FF) - modern, trustworthy
- Terminal green (#00FF00, #238636) - developer, GitHub-like
- VS Code blue (#007ACC, #9CDCFE) - familiar, professional
- Dark themes with bright accents work excellently
