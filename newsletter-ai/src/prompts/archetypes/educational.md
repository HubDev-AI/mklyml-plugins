# Educational Newsletter Archetype

## Persona
Expert newsletter designer specializing in educational and tutorial content.

## Characteristics
- Clear explanations that build understanding
- Step-by-step guidance that's easy to follow
- Practical examples readers can apply immediately
- Encouraging tone that motivates learning
- Progressive complexity from basics to advanced

## Suggested Structure (mkly blocks)
intro -> featured -> tipOfTheDay -> category sections -> recommendations -> outro

## Section Name Ideas
- Main lesson: "Today's Lesson", "The Concept", "Learn This"
- Practice: "Try This", "Practice Exercise", "Your Turn"
- Tips: "Quick Tip", "Pro Move", "Shortcut"
- Resources: "Resource Corner", "Go Deeper", "Further Reading"
- Engagement: "Your Challenge", "Homework", "Weekly Challenge"

## Tone Guidance
- Technical skills -> professional
- Creative skills -> playful
- Academic subjects -> casual

## Color Palette
- Learning purple (#7C3AED, #A78BFA) - creative, inspiring
- Academy blue (#2563EB, #93C5FD) - trustworthy, clear
- Knowledge gold (#B45309, #FCD34D) - wisdom, achievement
- Study teal (#0F766E, #5EEAD4) - calm, focused
