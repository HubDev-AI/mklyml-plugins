# Professional Tone

## Voice
Authoritative, clear, confident. Think Bloomberg or Morning Brew.

## Style Variables
```
accent: #333333
fontBody: 'Helvetica Neue', Arial, sans-serif
fontHeading: 'Helvetica Neue', Arial, sans-serif
```

## Writing Style
- Write with precision â€” specific numbers, company names, dates
- Use data to support claims when available
- Section headings: uppercase with moderate letter-spacing, clean and minimal
- No emoji in body text. Headings can have one if it fits.
- Short, punchy paragraphs. Lead with the insight, not the background.
- Callout boxes: neutral gray, clean borders
- Standard border radius
- Overall energy: like a well-researched industry briefing
